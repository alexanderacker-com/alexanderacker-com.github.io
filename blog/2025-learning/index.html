Signal Processing

Since I've been playing with audio processing for fun and often am adjacent to Computer Vision applications at work I've been interested in signal processing.

In general the skill of analyzing data in the frequency domain seems to help one gain intuition for abstract similarities between audio, images or other types of data especially in the time domain.

Resources

  • Mike X Cohen: applied courses for both signal processing, image processing and fourier transform
  • Iain Explains: applied engineering examples of fourier transform

Linear Algebra

I want to improve my intuitions for Linear Algebra and advance into more applied topics. I remember the basics from my undergraduate studies but want to get to more advanced topics which prove to be useful for Signal Processing, Computer Vision, ML and Statistics.

Resources

  • MIT 18.06 Linear Algebra Lectures
  • Linear Algebra Done Right1
  • MIT 18.065: Matrix Methods in Data Analysis, Signal Processing, and Machine Learning

Statistics

A similar story as with Linear Algebra here. I think however with Statistics I could probably take a more theoretical approach as the courses from my undergrad were very applied and I believe my intuition for a topic is usually better when I take a more rigorous approach. Especially given this is second pass through the topic it usually makes sense to try formalizing the concepts.

Resources

  • Statistical Inference Second Edition2
  • All of Statistics3
  • Statistics for Mathematicians: A Rigorous First Course4

Computer Vision
